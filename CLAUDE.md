# CLAUDE.md

Guidance for Claude Code when working in this repository.

## Project
- `go-app`: a starter Go project.
- Module: `github.com/mdelaiglesia/go-app`
- Entry point: `main.go`

## Common commands
- Run: `go run .`
- Build: `go build -o go-app .`
- Test: `go test ./...`
- Format: `gofmt -w .`
- Vet: `go vet ./...`

## Conventions
- Standard Go formatting (`gofmt`).
- Keep the module path in sync with the repo location if you move/rename it.
